## 简介

1.这是基于ThinkPHP3和Coreseek 3.2.14 开发的一套搜索管理系统。

2.本软件，源码完全开放。

## 安装说明

打开phpMyAdmin

a. 在mysql数据库，新建数据库searchsys

b. 将searchsys.sql 导入 searchsys

c. 打开 源码文件目录 Application/Common/Conf/config.php 修改数据库信息

d. 安装 coreseek 3.2.14

地址说明 

比如 我的网站根目录是 d:/www 我的源码放在 d:/www/searchsys 那么前台访问地址就是

http://localhost/searchsys

后台访问地址：

http://localhost/searchsys/Admin/Index/index.html

后台用户名 admin 和 密码 admin123
